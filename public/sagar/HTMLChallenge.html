Welcome To Andreas08!

I get a lot of request about new open source templets, and this one was created as a response to one of those requests. This template is based on the design of my own website, with only small changes (hopefully improvements!) from my own design. among the changes, it is worth to note that this template does not use any images - and browser-based font resizing is fully supported. Other than that, there are no special features or extras, only valid and well-structured XHTML 1.1 and CSS2 code.

Current template version: 1.0 (November 28, 2005).
